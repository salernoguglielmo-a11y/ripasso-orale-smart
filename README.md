# ripasso-orale-smart

Web app per il ripasso dell'esame orale da avvocato con focus su:
- diritto penale
- diritto civile
- procedura penale

## Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## Funzioni
- dashboard iniziale
- filtro per materia e stato
- ricerca full text
- doppia modalità di ordinamento: ordine logico o priorità
- simulazione orale con timer
- tracciamento locale del ripasso
- dataset strutturato per materia

## Avvio locale
```bash
npm install
npm run dev
```
